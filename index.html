Projects

Circumgalactic Dust Extinction in KiDS-DR4

Developed a Python pipeline to detect and model faint CGM dust extinction signals using multi-band survey data and simulations (MICE2).

Detection Bias in Cosmic Shear

Designed detection probability function using GalSim and Source Extractor to model blending-induced biases in shear surveys.

Near-IR Imaging of Radio Galaxy Hosts

Reduced and analysed LBT near-IR imaging of radio galaxy hosts using IRAF and GALFIT.
